    Assignment #6

    • Due Monday by 11:55pm
    • Points 20
    • Available after May 8 at 8am

    Objectives :

    • Using a FTP client to manage files and directories on a remote host.
    • Make a first web page.


    I - Download and Install FileZilla... carefully! How?

    1. Download the FileZilla program (latest version) from Filezilla Download   (Links to an external site.)Links to an external site.
    •  (Links to an external site.)Links to an external site.WARNING:  only click on ONE DOWNLOAD button that looks like this:
    •   or
    • . You may be redirected to another page.  After that, do not click any other buttons... just wait !
    • Save the download file on the desktop of your home computer.
    • Double-click it. Click OK on all dialog boxes - unless they are going to install extra software. When asked if you want a secure installation, say 'No'. It is now installed on your computer.

     

    II - Anonymous FTP
    Start Filezilla. Here's how to adjust settings for anonymous FTP. For this type of connection you DO NOT NEED an account on the ftp host.

    1. Choose ' Edit ' menu, then ' Settings ', then click on ' Firewall ' (you shouldn't have to do this in the latest version of Filezilla!)
    2. select ' Passive Mode ', then close Settings
    3. Make sure the QuickConnect toolbar is visible. Choose ' View ', then ' QuickConnect Bar '.
    4. Open the ftp site at the host ftp.hq.nasa.gov (other settings: anonymous for user, your email address for password, port number 21)
    5. Navigate to the pub/pao/movies directory
    6. Download rover_tiny.mov
    7. You can try to open this to see what it is!
    8. Quit Filezilla

     

    Try another ftp site. This one has meteorological data.

    1. Open the ftp site at data-out.unavco.org
    2. Navigate to the pub/rinex directory
    3. Download any file, preferably larger than 100kB but less than 5MB
    4. Quit Filezilla

     

    III - Upload Files to compass server
    At the bottom of this page is a listing of your login names for the compass server. Your password is your 9-digit student # . You will need this now.

    1. Start FileZilla. Create a ' New Site ' named compass . Fill in the following values:
      1. Host: compass.northseattle.edu
      2. Servertype: SFTP
      3. Port #: 22
      4. Logontype: Normal
      5. User: (enter your lower-case login name from list below)
      6. Password: (enter your 9-digit student number without the hyphen symbols )
    2. Click ' Connect '
    3. You should see something displayed in the right-hand pane. Navigate to your own folder. It is probably /home/students/yourloginname
    4. Right-click on the folder named homework , then delete it.
    5. Double-click on the folder named public_html . This is where all web pages and public files will be stored.
    6. Create a directory named images. ( right-click in the right-hand pane )
    7. Create a directory named documents.
    8. Create a directory named assignments.
    9. Upload the following files from your computer into the assignments directory in your remote directory:
      1. any 2 homework assignments that you have used for this class
      2. this web page ( the one you are looking at right now!)
    10. Upload the following files from your computer into the documents directory of your remote directory:
      1. the TWO files that you previously downloaded via anonymous FTP in Step II
    11. Upload the following files from your computer into the images directory in your remote directory:
      1. the equation from the instructor's web page for the MidTerm Test
      2. Any clip art image that you can find in WORD ( you can just paste it to the desktop! )
      3. The image of a computer that is at the top of this page

      NOTES:

      • A web page is a text file, so you can create it using NotePad on any Windows computer.
      • A web page should be constructed from properly arranged HTML tags. See the examples and notes for more help.

      NOTES TO MAC USERS:

      You can create web pages using TextEdit on the Mac, but...

      • be sure to choose 'Format' then 'Plain Text' format before you save it. Also disable Smart Quotes, See screen shots below:
      • Or download and install TextWrangler (Links to an external site.)Links to an external site.
    12. Upload a web page into your ' home ' directory. That is the directory that contains the folders named assignments , images , and documents. This web page should be named assignment6.html and contain your name and a paragraph that describes what you are planning to do in your career. It should have at least 3 style changes and a centered title. When correctly uploaded, it should be visible from any browser at http://compass.northseattle.edu/~loginname
    13. Quit Filezilla.
